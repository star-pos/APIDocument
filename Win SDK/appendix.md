## 返回码列表

| **返回码** | **值** | **错误描述** |
| :--- | :--- | :--- |
| RET\_API\_WAIT\_AUTH | 1 | 接口执行结果-等待授权 |
| RET\_SUCCESS | 0 | 请求成功 |
| RET\_UNKOWN | -1 | 未知错误 |
| RET\_PARAM\_INVALID | -2 | 参数不合法 |
| RET\_HTTP\_FAILED | -3 | 网络错误 |
| RET\_RESPONSE\_INVALID | -4 | 服务端返回信息不符合约定 |
| RET\_API\_FAILED | -100 | 接口执行结果-交易失败 |
| RET\_API\_UNKNOWN | -102 | 接口执行结果-交易未知 |
