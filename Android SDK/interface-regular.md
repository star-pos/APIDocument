# 接口说明

## 名词解析

| **名词** | **含义** |
| :--- | :--- |
| 商户主扫 | 商户扫描客户的付款二维码进行收款 |
| 客户主扫 | 客户扫描商家生产的二维码付款 |
| 退款 | 支付成功的订单才能进行退款 |
| 查询订单 | 查询某一笔订单的信息 |

## 接口约束

| **符号** | **含义** |
| :--- | :--- |
| 0 | 可选项 |
| 1 | 数字1，代表必须且只能填1项。 |

* 文档表格定义的字段，请准确写入请求参数中
* 使所有的value不能用字符串null和NULL赋值
* SecretKey值为商户密钥
